# ☁️🔥 職場情緒行事曆 (Mood Calendar)

**👉 [點擊這裡立即體驗：職場情緒行事曆 ](https://14sail.github.io/mood-calendar/) 👈**

一個專為上班族與團隊打造的職場情緒紀錄、量化統計與社交互動平台。使用者可以一鍵切換「Happy（泡泡基地）」與「Angry（充滿挑戰的一天）」雙模組主題，紀錄每日工作心情、統計正負能量，並能與好友雙向連線互動、互看心情日誌與留言打氣！

## 🚀 線上展示與部署
本專案為全前端靜態網頁架構，整合 **Firebase (BaaS)** 雲端服務，非常適合直接免費部署至 **GitHub Pages**。

---

## ✨ 核心特色功能

- **🎨 雙色調情緒主題切換**
  - **Happy 模式 (☁️ 泡泡基地)**：溫和明亮的藍色調，陪伴你記錄充滿成就感、快樂與順心的職場時光。
  - **Angry 模式 (💢 充滿挑戰的一天)**：深色沉浸式主題搭配醒目警示紅，優雅釋放工作中的無奈、心累與崩潰壓力。
- **📅 動態行事曆面板**
  - 整合當月日期，支援跨月份瀏覽與即時動態渲染。
  - 日曆格子同步顯示自己與好友發布的心情小圖示與自訂暱稱。
  - 自動鎖定未來日期，僅允許補登或撰寫今日以前的職場日記，防範預防性倒垃圾。
- **📊 量化情緒統計卡片**
  - **本月統計**：即時動態加總計算當月「快樂總合」與「生氣總合」。
  - **年度累計**：綜觀全年度的「快樂能量」與「怒火值/挑戰值」，將抽象情緒轉化為具體數據趨勢。
- **🔐 Firebase 第三方快速登入**
  - 整合 Google 帳號授權驗證，一鍵登入，保障個人資料隱私與獨立性。
  - 支援完善的個人化設定：可隨時自訂專屬的**個人暱稱**與**心情 Emoji 頭像**。
- **👯 雙向自動同步好友系統**
  - 透過暱稱精準搜尋並添加好友。
  - 內建**雙向自動同步邏輯**，任何一方送出添加，雙方即刻互為好友。
  - 行事曆同步共享：可在同一個日曆畫面上無縫查看所有好友的公開心情。
- **💬 互動對話式留言板**
  - 點擊日曆上的任何心情紀錄卡片，即可彈出檢視詳細故事。
  - 內建即時留言功能，對話框採用對稱式氣泡與高對比色調設計（生氣模式亦經過色彩對比優化），讓團隊或好友能即時取暖、吐槽或給予安慰。

## 🛠️ 技術疊層 (Tech Stack)

- **前端核心**：HTML5, CSS3 (變數驅動主題控管、全自動 RWD 響應式排版), JavaScript (ES6+ 異步非同步、原生 DOM 操控)
- **後端雲端服務 (BaaS)**：Firebase v10
  - **Firebase Authentication**：安全可靠的用戶帳號身分驗證。
  - **Cloud Firestore**（無伺服器分散式資料庫）：
    - `users`：儲存用戶暱稱與自訂 Emoji 資訊。
    - `users/{uid}/friends`：儲存雙向好友關係對應關係清單。
    - `records`：儲存每日情緒標題、詳細內容、等級指數及發布時間戳。
    - `records/{recordId}/comments`：建立獨立的巢狀留言板數據流。
